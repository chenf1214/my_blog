### 一次搞定位运算
先说下我们需要提前知道的一些小点
1. 10进制转二进制
已知十进制数 12 转为二进制是 1100 为什么？
简单的记忆办法是 12 / 2 每一步的余数是 0 0 1 1 
然后倒叙 就是 1 1 0 0

2. 二进制数转十进制
还是二进制数 1100 为什么转十进制是 12
简单的记忆办法是 我们翻过来看 0*2^0 + 0*2^1 + 0*2^2 + 0*2^3
也就是 0 + 0 + 4 + 8 = 12

ok 现在你已经知道基本的二进制和十进制了

### 位运算
1. << 左移
ps：一端时间以来，总觉得写什么 << 什么  >> 还有什么 ~ 这种代码的人很吊，奈何自己还不太懂
左移的运算规则是：各二进位全部左移若干位，高位丢弃，低位置补0
例如 12 << 2 = 48

```js
0000 0000 0000 0000 0000 0000 0000 1100     // -> 12 
0000 0000 0000 0000 0000 0000 0011 0000     // 12 << 2 =  0 + 0 + 0 + 0 + 2^4 + 2^5 = 16 + 32 = 48
```
惯例：如果真的到这里其实没啥意思，
发现规律 如果左移2 就是 12 * 2^2 = 48 
也就是左移几位就是 原数乘以2的几次幂
下次你在看什么左移2 的直接乘以四 ok？

2. >> 右移
运算符规则：各二进位全部右移动若干位，高位补0，低位丢弃
例如 12 >> 2 = 3

```js
0000 0000 0000 0000 0000 0000 0000 1100     // -> 12 
0000 0000 0000 0000 0000 0000 0000 0011     // 12 >> 2 =  1*2^0 + 1*2^1 = 1+2 = 3

0000 0000 0000 0000 0000 0000 0000 0111     // -> 7 
0000 0000 0000 0000 0000 0000 0000 0001     // 7 >> 2 =  1
```
惯例：如果真的到这里其实没啥意思，
发现规律
规律一：如果右移动2就是 12 / (2*2) = 3
17 / (2*2) = 4.25 好 你别管他余数是多少，他就是 4 了
7 / (2*2) = 1.75 好  你别管他余数是多少，他就是 1 了

规律二：右移动2不是丢了二个么
去看我们的二进制 偶数不用管 偶数后面是0 0 奇数 后面是1 1扔掉了
ok 那好，我们一步一步来，
先移动一位 （7-1）/ 2 = 3
在移动一位  （3-1）/ 2 = 1 好就是1了
先移动一位 （17-1）/ 2 = 8
在移动一位  8 / 2 = 4 好就是4了
